# Dotfiles

My Ubuntu dotfiles.

## Installation

using curl:
```sh
bash -c "$(curl -fsSL https://raw.github.com/jfgodoy/dotfiles/master/bin/dotfiles.sh)" && source ~/.bashrc
```

using wget:
```sh
bash -c "$(wget -qO- https://raw.github.com/jfgodoy/dotfiles/master/bin/dotfiles.sh)" && source ~/.bashrc
```
